Aide mémoire des sélecteurs CSS déjà vus

  • * sélectionne TOUS les éléments de la page html
  • .ma-jolie-nav sélectionne tous les éléments ayant le nom de class "ma-jolie-nav"
  • #mon-unique-id sélectionne tous les éléments ayant le nom de mon id "mon-unique-id"
  • div, article, aside sélectionne toutes les balises <div>, <article> ET <aside>
  • nav a sélectionne les liens <a> inclus/imbriqués dans la balise <nav>
  • nav a:hover sélectionne tous les liens de la nav et active une animation lors du survol de la souris

Propriétés CSS utilisées dans cette page

Décrivez leur usage et leurs valeurs possibles

  1. background: permet de modifier le fond de l'élément;
    EX: background: linear-gradiant (to left, #333, #fff);
      Les différents éléments utilisable dans le cadre de background:
    • red
    • #333
    • rgba(0,50,100,0.5)
  2. color: permet de modifier la couleur de la police;
    EX: color: lightblue;
      Les différents éléments utilisable dans le cadre de color:
    • red
    • #333
    • rgba(0,50,100,0.5)
  3. float: permet de fixer un élément du body, dans un emplacmeent choisit ceci est appellé un "effet flottant"
    EX: float: left;
      Les différents éléments utilisable dans le cadre du float:
    • right
    • left
    • inherit
  4. font-family: petmet de modifier la police de charactère;
    EX: font-family: 'Courier New', Courier, monospace;
      Les différents éléments utilisable dans le cadre de font-family:
    • 'Courier New', Courier, monospace
    • 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
    • 'Trebuchet MS', 'Lucida Sans Unicode', 'Lucida Grande', 'Lucida Sans', Arial, sans-serif
  5. font-size: petmet de modifier la taille de la police;
    EX: font-size: 1.5em;
      6. Les différents éléments utilisable dans le cadre de font-size:
    • 20px
    • 2em
    • small
    • ......
  6. height: petmet de modifier la hauteur de l'élément (block et non inline)
    EX: height: 8em;
      7. Les différents éléments utilisable dans le cadre de height:
    • 200px
    • 50%
    • 20em
  7. left: permet de positionner un élément (block) depuis la gauche de la fenêtre
    EX: left: 50px;
      8. Les différents éléments utilisable dans le cadre de left:
    • 50px
    • 20%
    • 10em
  8. line-height: permet de rajouter une hauteur à chaque interligne
    EX: line-height: 1%;
      9. Les différents éléments utilisable dans le cadre de line-height:
    • 50px
    • 20%
    • 10em
  9. margin: petmet de rajouter des marges extérieures
    EX: margin:5px;
      10. Les différents éléments utilisable dans le cadre de margin:
    • 50px
    • 20%
    • 10em
  10. padding: petmet de rajouter des marges intérieures
    EX: padding:5px;
      11. Les différents éléments utilisable dans le cadre de padding:
    • 50px
    • 20%
    • 10em
  11. position: position de l'élément dans la page (on n'a utilisé que la valeur (fixed) pour l'instant)
    EX: position: absolute;
      12. Les différents éléments utilisable dans le cadre de position:
    • absolute : par rapport à la page
    • relative
    • fixed : par rapport à la fenêtre
  12. text-align: alignement du texte
  13. text-decoration: changement des valeurs héritées (ici le souligné qui est standard pour les liens sous tous les navigateurs est supprimé)
  14. top: position à partir du haut de la fenêtre
  15. transition: temps de transition lors d'une animation (ici sur les liens de la barre de menus)
  16. width: largeur de l'élément par rapport à la taille de la fenêtre si on indique une valeur relative (%, em, ...)
